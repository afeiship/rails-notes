# api framework


## Rails::API
你已有了Rails的Web程序
但你不希望用Rails的ActionController去实现这个API
而是用个更加简单快速的方式

## Sinatra
Sinatra更加独立，服务范围更明确，与其他应用的共享的逻辑也更少
不仅仅用于JSON API的建立(易扩展成web应用)


## Grape
你有大型的内部API程序
你需要一个专注处理JSON API的先进框架
